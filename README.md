# Ticket Management System

A console-based Helpdesk Ticket System built using Python.  
The system allows users to create, view, and update support tickets with persistent JSON storage.

## Features

- Create a new support ticket
- View all existing tickets
- Close tickets
- Store tickets in a JSON file for persistence

## Technologies

- Python 3
- JSON file storage
